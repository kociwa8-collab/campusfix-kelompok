CampusFix

Platform pelaporan fasilitas kampus yang modern, efisien, dan responsif.

Laporkan kerusakan fasilitas kampus dengan mudah, pantau status perbaikan secara real-time, dan bantu menciptakan lingkungan kampus yang lebih nyaman.

Masuk ke Aplikasi Daftar Mahasiswa
Fitur Utama

Semua proses laporan kampus dibuat lebih jelas.

Mahasiswa membuat laporan, admin memantau, dan status perbaikan tetap terlihat.

01

Buat Laporan

Kirim lokasi, kategori, deskripsi, dan foto bukti fasilitas yang bermasalah.

 02

Pantau Status

Lihat perkembangan laporan dari menunggu verifikasi, diproses, sampai selesai.

 03

Dashboard Admin

Admin dapat melihat semua laporan, mencari data, memfilter status, dan mengubah progres.
Alur Kerja

Dari laporan masuk sampai ditangani.

1

Mahasiswa mengirim laporan

Data laporan tersimpan ke sistem dan muncul di riwayat mahasiswa.

2

Admin memverifikasi

Admin melihat laporan masuk di dashboard dan mulai proses penanganan.

3

Status diperbarui

Mahasiswa bisa memantau perubahan status laporan secara transparan.
